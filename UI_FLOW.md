## Color Scheme

- **Primary Blue**: `#007AFF` - Buttons, links, active states
- **Background**: `#f5f5f5` - Main background
- **Card White**: `#ffffff` - Cards, containers
- **Text Dark**: `#333333` - Primary text
- **Text Medium**: `#666666` - Secondary text
- **Text Light**: `#999999` - Tertiary text
- **Border**: `#e0e0e0` - Dividers and borders
- **Success**: `#34C759` - Success states
- **Warning**: `#FF9500` - Warning states
- **Danger**: `#FF3B30` - Delete, sign out

## Typography

- **Large Title**: 28pt, Bold
- **Title**: 24pt, Bold
- **Heading**: 20pt, Bold
- **Subheading**: 18pt, Semibold
- **Body**: 16pt, Regular
- **Caption**: 14pt, Regular
- **Small**: 12pt, Regular
- **Tiny**: 10pt, Regular

## Interactions

### Sign In Flow
1. User sees sign-in screen
2. Taps "Sign in with Google"
3. Google authentication modal appears
4. User selects account and authorizes
5. Redirected to Home screen
6. Profile auto-created in Firestore

### Create Activity Flow
1. Tap "+ Create" on Activities tab
2. Fill in activity details
3. Tap "Create Activity"
4. Date validation runs
5. Activity saved to Firestore
6. "Organizer" badge awarded
7. Redirected back to Activities list

### Join Activity Flow
1. Browse activities
2. Tap on an activity card
3. View activity details
4. Tap "Join Activity"
5. User added to participants
6. "First Steps" badge awarded (if first activity)
7. Can now access activity chat

### Chat Flow
1. Join an activity
2. Navigate to activity detail
3. Scroll to chat section
4. Type message in input
5. Tap "Send"
6. Message appears immediately (real-time)
7. Other participants see message

### Profile Editing Flow
1. Go to Profile tab
2. Tap "Edit"
3. Modify name and select interests
4. Tap "Save"
5. Changes synced to Firestore
6. UI updates immediately
